# UGS Validator Chrome Extension

Real-time JEXL validation for Unity Gaming Services configurations.

## Installation

1. Open Chrome and navigate to `chrome://extensions/`
2. Enable "Developer mode" (toggle in top right)
3. Click "Load unpacked"
4. Select the `extension` folder

## Features

- Real-time JEXL syntax validation
- UGS/Wisdom parameter validation
- Intent verification
- Inline error/warning indicators
- Floating validation overlay

## Development

The extension injects into UGS pages and enhances the UI with validation feedback.
